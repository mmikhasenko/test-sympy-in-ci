SymPy in Pluto

The repo test the Sympy installation with Pluto notebooks in CI.

It used to be sufficient to add empty the PYTHON environment variable to CI yaml file.

        runs-on: ubuntu-latest
        env:
            PYTHON: ''

However, the notebooks appear to fail. E.g. https://mmikhasenko.github.io/rescattering-3pi-2023-001/crosscheck/N-001-cartesian-pions.html

Using this repository, I found out that the probkem is caused by another Project.toml file in a subfolder. Naively, it should not affer the top level file processing, but it does.
